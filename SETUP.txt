TESTING ENGINE — SETUP GUIDE (Windows)
=======================================

STEP 1 — Clone the repo
------------------------
  git clone https://github.com/VICKYPATIL2525/test-engine.git
  cd test-engine


STEP 2 — Create and activate virtual environment
-------------------------------------------------
  python -m venv venv
  venv\Scripts\activate


STEP 3 — Install dependencies
------------------------------
  pip install -r requirements.txt


STEP 4 — Set up your .env file
--------------------------------
  copy .env.example .env

  Open .env and fill in the credentials shared by your team lead:

  OPENAI_API_KEY=...
  AZURE_OPENAI_ENDPOINT=...
  AZURE_OPENAI_VERSION=2024-12-01-preview
  AZURE_OPENAI_DEPLOYMENT=gpt-4.1-mini
  GITHUB_TOKEN=...
  REPO_URL=https://github.com/<org>/<repo>.git


STEP 5 — Pull commit data from your target repo
-------------------------------------------------
  python git_extractor.py


STEP 6 — Add your HTML test reports
-------------------------------------
  Create a folder called html-reports in the project root.
  Place your Spark HTML report files inside it.

  html-reports/
      run1_TestRunReport.html
      run2_TestRunReport.html


STEP 7 — Run the app
----------------------
  streamlit run strapp.py

  Opens in browser at: http://localhost:8501


TROUBLESHOOTING
---------------
  "No data found"          -> Run step 5 first
  "Missing credentials"    -> Check your .env file
  "ModuleNotFoundError"    -> Run step 3 again with venv active
  "No test reports found"  -> Add HTML files to html-reports/ folder
